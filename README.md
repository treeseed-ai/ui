# @treeseed/ui

Reusable TreeSeed UI primitives, theme utilities, React components, and an Astro component sandbox.

```sh
npm install @treeseed/ui
```

Import base styles once in an app shell:

```ts
import '@treeseed/ui/styles/tokens.css';
import '@treeseed/ui/styles/theme.css';
import '@treeseed/ui/styles/ui.css';
import '@treeseed/ui/styles/forms.css';
```

Run the local sandbox:

```sh
npm run dev
```
